[reactboilerplate-sg
](## React Boilerplate with Server side rendering

```
#directory structure
|--public
    |--css
       |--images
    |--scss
|--src    
    |--client
        |--assets
        |--components
        |--layout
        |--utils
    |--server
        |--helpers
```

#### Setup

Clone the repo:
```
cd ./<project directory>
git clone https://bitbucket.org/studiosg/reactboilerplate.git .
npm install
npm start
```

#### Production build
```
npm install
npm run build
```
Application will be served on http://localhost:3005
)
